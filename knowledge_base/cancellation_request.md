# Cancellation Request

This category covers requests to stop, terminate, close, or cancel an order, account, service, booking, or subscription.

Examples include:

- Cancel my subscription.
- Stop my recurring service.
- Close my account.
- Cancel an order before shipment.
- Terminate my membership.
- Unsubscribe me from the service.

Do not use this category when the customer's primary request is to receive money back. In that case, use refund request.
