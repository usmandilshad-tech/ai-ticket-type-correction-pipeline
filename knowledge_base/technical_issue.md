# Technical Issue

This category covers cases where a product, device, application, account, or service is not working correctly.

Examples include:

- The device is not working.
- The product is making strange noises.
- The application crashes.
- The customer cannot log in.
- Password reset does not work.
- An error message appears.
- The system behaves intermittently.
- A hardware component appears damaged.
- The page fails to load.
- The software contains a bug.

Do not use this category when the customer is only asking about features, compatibility, or setup and no malfunction has been reported.
